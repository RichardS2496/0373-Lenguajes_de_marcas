# ShieldSecurity – Web Corporativa

Proyecto desarrollado para el módulo **Lenguajes de Marcas y Sistemas de Gestión de Información (0373)** del ciclo DAW.

**ShieldSecurity** es una empresa ficticia de ciberseguridad que ofrece servicios de monitorización, detección y neutralización de amenazas digitales para autónomos, pequeñas empresas y usuarios particulares.

---

## Páginas incluidas

| Página | Archivo | Descripción |
|---|---|---|
| Inicio | `index.html` | Landing page principal con secciones de servicios, nosotros y contacto |
| Blog | `blog.html` | Listado de artículos del blog |
| Post | `blog/post_1.html` | Artículo individual del blog |

La página de inicio incluye las secciones de navegación: **Inicio**, **Blog**, **Servicios**, **Nosotros** y **Contacto**.

## Abrir en local

No se requiere servidor ni instalación de dependencias. Basta con abrir los archivos HTML directamente en el navegador:

1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` con tu navegador preferido (doble clic o arrastrar al navegador).
3. Navega entre las páginas usando el menú de navegación.

> Para una experiencia completa con rutas relativas funcionando correctamente, se recomienda usar la extensión **Live Server** de VS Code o cualquier servidor local equivalente.
