Aclaración previa
La web enlazada arriba no forma parte del proyecto desarrollado para el ciclo DAW. Se trata de un portafolio personal que construí con anterioridad al ciclo, pensado como espacio de presentación propia, no como entrega técnica.
Al figurar el portafolio como entregable opcional / básico dentro del módulo de empleabilidad, he decidido reutilizar esta web en lugar de crear una nueva desde cero. Cumple con el objetivo del enunciado: ofrecer una estructura mínima donde mostrarme como persona y como aprendiz de desarrollo.

Toda la información técnica del proyecto del ciclo (descripción, capturas, repositorio, aprendizajes) está en el README del proyecto ShieldSecurity, no aquí. El portafolio y el proyecto son piezas distintas y se mantienen separadas a propósito.

Qué es este portafolio
Una pequeña web personal que sirve como tarjeta de presentación. Muestra quién soy, qué me interesa y dónde encontrarme, sin entrar en el detalle técnico de los proyectos. Es un espacio estable, separado del repositorio académico, al que puedo enlazar desde cualquier sitio.
Qué se puede ver en ella

Una breve presentación personal.
Información sobre mí, mis intereses y mi recorrido.
Datos de contacto.

Por qué la incluyo como portafolio del módulo

Ya está desplegada y operativa en Netlify, accesible desde cualquier navegador.
Refleja una versión previa de mí mismo como aprendiz, anterior al ciclo.
Encaja con la idea de "portafolio básico" que plantea el enunciado: una estructura mínima donde verme.
Permite contrastar mi punto de partida con el nivel actual, que queda recogido en el proyecto del ciclo.
